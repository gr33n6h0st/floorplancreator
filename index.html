The Tool: A "Lightweight" CAD Suite

Floorplancreator.net is a cloud-based CAD tool that effectively bridges the gap between rough hand-sketching and heavy industrial software like AutoCAD[cite: 187]. It is built for BYOD (Bring Your Own Device) environments, allowing students to start projects on Chromebooks and finish them on mobile phones[cite: 188].

Smart-Snap System

Drag-and-drop walls and windows with automatic alignment and joint healing. Supports exact metric and imperial dimensions for scalable technical drawings[cite: 191].

Instant 3D Visualization

One-click transformation from 2D blueprint to a navigable 3D model, allowing students to verify sightlines and ergonomics instantly.

Pedagogical Rationale

This tool combats "Creation Fatigue" by removing the mechanical barrier of re-drawing layouts by hand[cite: 193, 194]. It shifts the cognitive load from drawing lines to solving design problems[cite: 198].

ONTARIO CURRICULUM ALIGNMENT

  • Expectation A1: Develop and evaluate solutions via rapid prototyping[cite: 195, 196].
  • Expectation B1: Mastering technical concepts and standard architectural symbols[cite: 199, 200].

"Floorplancreator.net is not just a drawing app; it is a feedback engine." [cite: 213]

Classroom Applications

A: Tiny Home Challenge

Uses Live Area Calculation to teach spatial budgeting. As students drag walls, square footage updates in real-time, providing immediate scaffolding for math integration[cite: 202, 203, 204].

B: Accessibility Retrofit

Students renovate JPEG floor plans to meet AODA standards. They use 3D walkthroughs to test wheelchair maneuverability and user experience[cite: 206, 207, 208].

C: Collaborative Planning

Mimics industry BIM (Building Information Modeling). Groups use "Project Sharing" to work on one master file simultaneously[cite: 209, 210, 212].
