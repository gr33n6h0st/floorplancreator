How to Use This Tool in the Classroom

Floorplancreator.net is a browser-based CAD tool. To achieve the results shown in this portfolio, the user follows this specific operating procedure:

  • 01. Drafting

    Select wall elements from the sidebar library. Drag them onto the canvas where they "snap" together automatically to form closed rooms.

  • 02. Dimensioning

    Click any wall to reveal the input field. Type exact metric measurements (e.g., 3500mm) to resize the structure to scale.

  • 03. Visualizing

    Press the "3D" button in the top toolbar to instantly render the 2D plan into a walkable 3D model for verification.

01 / Technical Workflow

Fig 1.1 / Main Interface

The Smart-Snap System

As demonstrated in Fig 1.1, the interface features an intelligent snapping engine. When a student drags a new room shape near an existing wall, the software automatically heals the joints. This removes the frustration of manual trimming found in AutoCAD, allowing Grade 11 students to focus on layout logic rather than line weight.

Input Rigor & Standards

The tool satisfies Expectation B1 by requiring precise data entry. Students cannot simply "sketch"; they must input specific dimensions, mirroring the rigor required in architectural technology post-secondary programs.

02 / Feedback Engine

Fig 2.1 / 3D Verification

Instant 3D Visualization

The defining pedagogical feature is the ability to toggle between abstract 2D plans and concrete 3D views. This allows for immediate Ergonomic Verification. A student can place a window in 2D, switch to 3D, and immediately see if the sightlines are blocked, fulfilling Expectation A1 (testing and refining solutions).

Scenario B: AODA Retrofit

Accessibility
Fig 3.2 / Compliance

Context: Renovating a bathroom for wheelchair accessibility.
Application: Students verify a 1500mm turning radius using the circle tool. The 3D walkthrough feature is then used to simulate the user experience, building empathy and ensuring compliance with Ontario's AODA standards.

Scenario C: Collaborative BIM

Industry Prep
Fig 3.3 / Collaboration

Context: A group design project for a community centre.
Application: Leveraging the cloud-based nature of the tool, students share project files URL links. This mimics Building Information Modeling (BIM) used in industry, where multiple stakeholders review a "living" digital model.
