01 / The Interface

Fig 1.1 / Main Workspace
Feature A: Smart Snapping

The workspace shown above features an intelligent "healing" engine. Unlike standard line drawing, walls automatically join when dragged near each other. This allows for rapid prototyping of layouts without manual trimming.

Feature B: Metric Rigor

As seen in the sidebar controls, the tool requires specific metric inputs. This enforces Expectation B1, moving students away from artistic sketching and toward engineering-level precision.

02 / Visualization Engine

Fig 2.1 / 3D Mode
Verification Loop

The software allows instant toggling between the 2D plan (above) and this 3D view. This creates a feedback loop where students can immediately test the "livability" of their design, satisfying Expectation A1 (Solution Testing).
